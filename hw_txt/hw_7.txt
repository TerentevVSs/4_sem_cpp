1. Каким образом происходит передача информации между устройствами в сети Интернет?
Передача происходит по протоколам TCP и IP
TCP обеспечивает разбиение на пакеты и сборку данных из пакетов у полчателя
IP обеспечивает доставку данных между устройствами
пакет несет в себе:
    тправитель
    получатель
    данные
    число 255=в начале, которое при прохождении через роутер
    (промежуточное устройство, перенапрвляющее посылки) уменьшается на 1
    и если становится 0, пакет удаляется или же пакет доходит до получателя раньше

2. Почему передача больших объёмов данных
(например, архива с датасетом для машинного обучения)
по Интернету не задерживает обмен данными между другими устройствами в сети?
2.1 Так как существует много вариантов путей, котороми можно отправить данные,
то трафик можно распределить между несколькими путями
2.2 Данные передаются пакетами, поэтому остальным усройствам не придется ждать,
пока закончится передача всех данных между 2 пользователями
2.3 Обычно оптоволокнонные линии передачи данных содержат много волокон,
по которым можно передавать данные, что позволяет распараллелить передачу данных
